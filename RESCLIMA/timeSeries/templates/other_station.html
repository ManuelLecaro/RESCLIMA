Subir otras Estaciones climáticas

Resclima soporta el ingreso de datos desde estaciones HOBO-X y BloomSky, pero también permite la carga de datos desde otras estaciones, para lo cual los archivos deben tener el siguiente formato

Datos necesarios

Para subir un archivo se requieren al menos 4 lineas:

  • Primera línea: Contiene el número de serie de la estación
  • Segunda línea: Vacío
  • Tercera línea: Contiene el header del csv
  • Cuarta línea: Datos
# Nombre de header Descripción Unidad
